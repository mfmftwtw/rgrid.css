# rGrid.css

`rgrid.css` is a quick responsive grid system.

## Example

[http://var.tw/mfmftwtw/rgrid.css/examples/](http://var.tw/mfmftwtw/rgrid.css/examples/)

## Quick Start

### Include the file.

```html
  <link rel="stylesheet" href="rgrid.min.css">
```

## Documentation

### All devices

```html
<div class="rg-1"></div>
```

|                   | Class Name         |
| ----------------- | ------------------ | 
| width: 1%         | `rg-1`          |
| width: 100%       | `rg-100`        |
| visible           | `rg-block`      |
| hidden            | `rg-none`       |
