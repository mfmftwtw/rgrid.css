# rGrid.css

rGrid.css is a quick responsive grid system.

## Example

[http://var.tw/mfmftwtw/rgrid.css/examples/](http://var.tw/mfmftwtw/rgrid.css/examples/)

## Quick Start

### Include the file.

```html
<link rel="stylesheet" href="rgrid.min.css">
```

## Documentation

### All Devices

```html
<div class="rg-1"></div>
```

| Description | Class Name |
| ----------- | ---------- | 
| Width: 1%   | `rg-1`     |
| Width: 100% | `rg-100`   |
| Visible     | `rg-block` |
| Hidden      | `rg-none`  |
